Представлен режим игры "друг с другом". Игроки по очереди ставят на свободные клетки поля 3×3 знаки, нажимая левую кнопку мыши (один всегда крестики, другой всегда нолики). Первый ход делает игрок, ставящий крестики.

Цель игры - первым выстроить в ряд 3 своих фигуры либо по вертикали, либо по горизонтали, либо по диагонали, чтобы выиграть.
